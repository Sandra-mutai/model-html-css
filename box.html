box model

    Box model is used when dealing with design and layout,or we can refer to how HTML elements
    are modeled in browser and how dimension of those html elements are derived from css properties.

    A box model has the following properties
    (a)Margin
    (b)padding
    (c)height
    (d)float
    (e)width
    (f)clear
    (f)border
to calculate width
400px width
+ 20px padding
+ 2px border
+ 20px margin= 442px
